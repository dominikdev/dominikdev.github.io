12 Grid Responsive Layout

This is a lightweight responsive framework that i put together. I based it on the 12-grid system that Bootstrap uses. It was build with the intention of being extremly lightweight for use in projects that don't require the extensive features that Bootstrap provides. It classifies screen sizes using bootstraps xs - lg technique. As well their system of offsetting content boxes.

Content Box Classes

This table shows how each class displays at each screen resolution

bw-xs-* bw-sm-* bw-md-* bw-lg-*
Screen < 768px x 100% 100% 100%
768px <= Screen < 992px x x 100% 100%
992px <= Screen < 1200px x x x 100%
1200px < Screen x x x x
